# Hotel Review Sentiment & Authenticity Dashboard

## Project Overview

This project analyzes hotel customer reviews to identify sentiment patterns, review authenticity trends, and customer feedback behavior using Power BI and analytical techniques.

The dashboard provides interactive visual insights into:

* Positive vs negative sentiment distribution
* Genuine vs deceptive review patterns
* Sentiment score distribution
* Source-wise review analysis
* Customer sentiment vs review authenticity relationship

The project demonstrates the use of Business Intelligence and data analytics techniques for extracting actionable insights from review datasets.

---

# Objectives

* Analyze customer sentiment trends across hotel reviews
* Identify potentially deceptive review behavior
* Understand review authenticity distribution
* Visualize customer feedback patterns through interactive dashboards
* Generate business insights and recommendations using review analytics

---

# Tools & Technologies Used

* Power BI
* Power Query
* Data Cleaning & Transformation
* Sentiment Analysis
* Dashboard Design & Visualization

---

# Dashboard Features

* KPI cards for review metrics
* Interactive slicers and filters
* Sentiment distribution analysis
* Authenticity distribution visuals
* Source-wise review analysis
* Sentiment score histogram
* Key business insights section

---

# Key Insights

* Approximately 82% of reviews exhibit positive sentiment patterns
* Deceptive and genuine reviews show overlapping sentiment behavior
* Sentiment score distribution indicates strong positivity bias in customer reviews
* Analytical techniques help identify potentially deceptive customer review patterns

---

# Dataset Information

The dataset contains processed hotel review data with attributes related to:

* Review authenticity
* Review source
* Sentiment score
* Sentiment classification
* Review type
* Hospitality domain categorization
